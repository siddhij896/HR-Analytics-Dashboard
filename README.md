HR Analytics Dashboard

## Project Overview

This project focuses on analyzing Human Resources (HR) data to understand employee demographics, attrition trends, salary distribution, and workforce performance. The data was cleaned using Excel, analyzed using SQL, and visualized through an interactive Power BI dashboard.

## Objectives

* Analyze employee attrition and retention patterns.
* Understand workforce demographics.
* Evaluate department-wise employee distribution.
* Analyze salary trends and employee characteristics.
* Generate actionable insights through data visualization.

## Tools & Technologies

* Microsoft Excel (Data Cleaning)
* MySQL (Data Analysis)
* Power BI (Dashboard & Visualization)
* GitHub (Project Documentation)

## SQL Analysis Performed

* Total Employee Count
* Attrition Analysis
* Department-wise Employee Distribution
* Gender Distribution
* Job Role Analysis
* Salary Analysis
* Age Group Analysis
* Overtime vs Attrition Analysis
* Education Field Analysis
* Marital Status Analysis

## Dashboard Features

* KPI Cards (Total Employees, Attrition Count, Attrition Rate, Average Salary)
* Department-wise Analysis
* Gender Distribution
* Job Role Distribution
* Attrition Analysis
* Interactive Slicers and Filters

## Key Insights

* Identified departments with higher attrition rates.
* Analyzed salary distribution across departments.
* Evaluated workforce demographics and employee trends.
* Examined the impact of overtime on employee attrition.
